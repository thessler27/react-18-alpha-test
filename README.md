# React 18 Alpha Test

Mostly based on Dan Abramov's [implementation on code sandbox](https://codesandbox.io/s/kind-sammet-j56ro?file=/src/Comments.js).
## To Install

`yarn install`

## To Run Dev

1. `yarn start`
2. Navigate to `localhost:3006`

## To Run Prod

1. `yarn start:prod`
2. Navigate to `localhost:3006`.
